ОПИСАНИЕ
--------

Плагин «Topicextend» предназначен для расширения функционала создания топика

* Добавляет в меню пункт "Создать топик" для всех пользователей, в том числе и незарегистрированных
* Если незарегистрированный пользователь решит создать топик, то произойдет перенаправление
  на форму авторизации вместо страницы ошибки
* Добавляет атрибуты размеров и альтернативного текста для изображений.
* Добавлена фукция (модификатор Smarty) склонения существительных по правилам украинского языка
* Добавлена поддержка работы с jQuery
